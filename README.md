# 🧠 Glaucoma Detection using CNN

## 📌 Overview

This project aims to detect **Glaucomatous Optic Neuropathy (GON)** from fundus images using a Convolutional Neural Network (CNN).

The dataset contains **747 Deep Fundus Images (DFIs)** with annotations:

* GON+ (glaucoma)
* GON- (normal)

---

## 📂 Dataset Structure

* `Images/` → original fundus images
* `images_resized/` → resized images for training
* `Labels.csv` → metadata and labels

### 🏷️ Label Information

* **GON+** → Glaucoma
* **GON-** → Non-glaucoma
* **Quality Score** → Range 1–10

---

## 📊 Dataset Statistics

* Total images: **747**
* Glaucoma (GON+): **548**
* Non-glaucoma (GON-): **199**

---

## 🤖 Model

* Model: Convolutional Neural Network (CNN)
* Task: Binary Classification (GON+ vs GON-)
